# ChurnSense Analytics — Customer Churn Prediction Dashboard

> **Author:** Smeet Patel | M.Tech CSE, Delhi Technological University | 2026

An end-to-end predictive analytics dashboard that analyzes customer churn patterns and provides real-time churn risk predictions through an interactive Streamlit interface. The project combines data preprocessing, exploratory analysis, feature-driven modeling, and business-focused dashboarding.

## Core Features
- Interactive KPI dashboard for churn insights
- Customer-level churn risk prediction
- Data preprocessing and feature transformation
- Model training and evaluation workflow
- Plotly-based interactive visualizations
- Business-friendly analytics for retention use cases

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Plotly
- Pandas
- NumPy
- Seaborn
